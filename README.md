## 🚀 Usage

1️⃣ Clone the repo:

```sh
git clone https://github.com/Doise/angular-todo.git
```

2️⃣ Install dependencies:

```sh
npm install
```

3️⃣ Run the server:

```sh
ng serve --open
```
